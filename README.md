# 🪐 Orbit — iOS-Style Remote Controller for Windows

**Orbit** — это премиальный пульт дистанционного управления вашим Windows-компьютером прямо с экрана мобильного телефона, выполненный в стиле нативного iOS-приложения с эффектом стекломорфизма (Glassmorphism).

Проект работает напрямую в вашей локальной Wi-Fi сети (по протоколу HTTP), обеспечивая **минимальную задержку ввода (<5 мс)**, высокую скорость работы и полную независимость от облачных сервисов.

---

## ✨ Основные возможности

- 📱 **Интерфейс в стиле iOS**: Элегантная тема с эффектами размытия, скругленными углами и микроанимациями.
- 🖥 **Стриминг экрана в реальном времени**: Встроенный режим просмотра экрана ПК с поддержкой «виртуального сенсорного ввода» (абсолютное позиционирование курсора по касанию).
- 🖱 **Виртуальный трекпад**: Поддержка жестов прокрутки двумя пальцами, кликов левой/правой кнопкой и перетаскивания окон.
- ⌨ **Удаленный ввод текста**: Полноценный клавиатурный ввод с поддержкой служебных клавиш (Enter, Backspace, Esc и т.д.).
- 🔊 **Медиа-панель**: Регулировка уровня громкости звука Windows и управление воспроизведением (пауза, переключение треков).
- 🚀 **Быстрый запуск приложений**: Автоматическое сканирование ярлыков меню «Пуск» ПК и запуск программ (Chrome, Steam, Telegram и др.) в один клик.
- 🔒 **Криптографическая безопасность**: Защита соединений через веб-сокеты с проверкой подписей HMAC-SHA256.

## 🛠 Установка и запуск

### Шаг 1. Установка зависимостей агента (Python)
Для работы сервера управления на компьютере должен быть установлен Python 3.11+.

1. Перейдите в папку `agent`:
   ```bash
   cd agent
   ```
2. Создайте виртуальное окружение и установите пакеты:
   ```bash
   python -m venv venv
   venv\Scripts\pip install fastapi uvicorn pynput zeroconf pycaw pywin32 pillow qrcode websockets
   ```

### Шаг 2. Сборка веб-интерфейса (Node.js)
Для сборки фронтенда потребуется установленный Node.js.

1. Перейдите в папку `frontend`:
   ```bash
   cd ../frontend
   ```
2. Установите зависимости и соберите продакшн-версию:
   ```bash
   npm install
   npm run build
   ```
   *Результат сборки автоматически попадет в папку, откуда её будет раздавать Python-сервер.*

### Шаг 3. Запуск
Просто дважды кликните по файлу `start_agent.bat` в корневой папке проекта.
1. В открывшейся консоли отобразится **QR-код** и локальный IP-адрес сервера (например, `http://192.168.1.5:23810`).
2. Подключите ваш телефон к той же Wi-Fi сети, что и компьютер.
3. Отсканируйте QR-код камерой телефона или введите IP-адрес в браузере.

### Шаг 4. Добавление на экран «Домой» (PWA)
Чтобы Orbit выглядел как нативное приложение:
- **На iOS (Safari)**: Нажмите кнопку **«Поделиться» (Share)** ➡️ **«На экран "Домой"» (Add to Home Screen)**.
- **На Android (Chrome)**: Нажмите на троеточие в правом верхнем углу ➡️ **«Добавить на гл. экран»**.

## 🔌 Решение для Wake-on-LAN (Включение компьютера)

Поскольку Orbit работает в браузере мобильного телефона, по соображениям безопасности браузерная песочница запрещает отправлять сырые UDP-пакеты (требуемые для Wake-on-LAN) напрямую из веб-приложения. Кроме того, когда компьютер выключен, сервер агента не работает.

Для включения ПК с телефона используйте следующую связку:

### 1. Подготовка компьютера (настройка BIOS и Windows)
1. **Настройка BIOS/UEFI**: 
   - Перезагрузите ПК и зайдите в BIOS (клавиши Del или F2 при запуске).
   - Найдите раздел электропитания (APM / Power Management) и включите параметр **Wake-on-LAN**, **Power On By PCI-E** или **Resume by PCI-E Device**.
2. **Настройка Windows**:
   - Нажмите `Win + X` и выберите **Диспетчер устройств**.
   - Найдите вашу сетевую карту в разделе **Сетевые адаптеры**, кликните правой кнопкой мыши ➡️ **Свойства**.
   - Во вкладке **Управление электропитанием** отметьте галочками: *«Разрешить этому устройству выводить компьютер из ждущего режима»* и *«Разрешать вывод компьютера из ждущего режима только с помощью магического пакета»*.
   - Во вкладке **Дополнительно** включите свойство **«Включение по локальной сети после отключения»** (Wake on Magic Packet).

### 2. Способы включения с телефона

*   **Вариант А. Бесплатное мобильное приложение (Рекомендуется)**
    Скачайте на телефон специализированную утилиту для отправки WoL-пакетов:
    - **iOS**: Приложение **Wolow** (поддерживает удобные виджеты для главного экрана) или **iNet**.
    - **Android**: Приложение **Wake on Lan** (от Mike Webb) или **Ping & DNS**.
    *В приложении достаточно один раз указать MAC-адрес сетевой карты вашего ПК и его локальный IP. После этого компьютер будет включаться в один тап.*

*   **Вариант Б. Умная розетка (Альтернатива)**
    Самый надежный способ включения из любой точки мира, даже если домашняя сеть недоступна:
    1. В BIOS вашего ПК включите параметр **Restore on AC Power Loss ➡️ Power On** (автоматический запуск при подаче питания).
    2. Подключите системный блок к умной розетке (Xiaomi, Tuya, Sonoff и др.).
    3. Для включения ПК просто выключите и включите розетку через приложение умного дома или голосового помощника (Алису, Siri, Google).

## 📝 Разработчикам

Проект разбит на две части:
1. **`frontend/`**: Клиентское React 19 приложение на TypeScript с анимациями `framer-motion` и локальной базой данных `Dexie.js` (IndexedDB) для сохранения настроек.
2. **`agent/`**: Python FastAPI веб-сервер, эмулирующий действия мыши/клавиатуры с помощью `pynput` и регулирующий системные параметры Windows через `pycaw`/`pywin32`.
